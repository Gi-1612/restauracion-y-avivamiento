# Restauración y Avivamiento — Web App

Este es el proyecto completo, listo para publicarse en Vercel de forma gratuita.

Seguí la guía "Cómo publicar tu app" que te compartió Claude junto con esta carpeta para el paso a paso completo (crear cuenta de GitHub, subir el proyecto, conectar con Vercel).

## Si más adelante contratás un developer

Puede correr el proyecto en su computadora con:

```
npm install
npm run dev
```

Y generar la versión de producción con:

```
npm run build
```

## Contenido

- `src/App.jsx` — toda la app (pantallas de Inicio, Reuniones, Novedades e Historial)
- `public/` — el logo y los íconos de la app
- Configuración de Tailwind CSS y PWA (para que se pueda "agregar a inicio" en el celular)
